HiC3D-Viewer

HiC3D-Viewer is a user-friendly tool designed to help investigator visualize Hi-C data in 3D fashion. It is based on a the WebGL capabilities offered by modern browsers to dispplay 3D graphics without a need for thrird party plug-ins.

Main features

Easy to use

A user-friendly interface and no programming is needed

Pre-built genomes

HiC3D-Viewer comes with build 3D models for some published genomes

Build you own model

HiC3D-Viewer alow users to predict a genome 3D model for their own data

Custom annotation

Users can highlight genomic region throught an interactive Hi-C heatmap or by uploading bed file
